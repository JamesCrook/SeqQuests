Features

This is a prototype of the UI. These are the features below which actually do anything currently:


  • Search Results - The distilled results
  • API Reference - This gives the server endpoints
  • A Help Demo (WIP) - Help that tells you what different buttons do.
  • Bias Plot (WIP) - Will become helpful for biased sequences.
  • Domain Plot (WIP) - Will show domains, and alignments.
  • Help (at GitHub) - The documentation.
  • Job Queue - You can run the all-on-all search job from here, rather than via CLI.
