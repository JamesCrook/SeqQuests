# LCARS UI


There are two LCARS UIs, themed based on the StarTrek control panels. The control buttons are in a frame around the data display area. Often there are two displays, an upper one for the main data and a smaller lower one for fine selection of what to show.

----

## Pedagogical Protein Viewer

This is a basic verison that browses a small curated list of proteins, using the NGL protein structure viewer.

* Lower control panel buttons select which protein to view
* Upper control buttons set options for the view, such as ball-and-stick.

## Job Control Panel

This is for use with the [web server](./web_server.md)

When the web server is running, this can show queued search jobs and job progress. This allows monitoring of a search job that is running on a differentr computer.
