# Frontend Files

The frontend of SeqQuest is built using vanilla JavaScript and HTML.

* [app.js](../../static/app.js) - The main application logic.
* [lcars.js](../../static/lcars.js) - The LCARS UI framework logic.
* [stream.js](../../static/stream.js) - Logic for handling data streams.
* [monitor.js](../../static/monitor.js) - Job monitoring logic.
* [bias_analysis.html](../../static/bias_analysis.html) - Bias analysis tool.
* [connector.html](../../static/connector.html) - Connector tool.
* [tree_browse.html](../../static/tree_browse.html) - Tree browser.
* [view_3d_pair_demo.html](../../static/view_3d_pair_demo.html) - Demo for 3D pair viewing.
